# Pallets V2

A Python application developed to simplify pallet counting in a warehouse environment.
This project is the second version of my pallet counting application. It was refactored using Object-Oriented Programming (OOP) to improve code organization, readability, and maintainability.

## About:
This is an improved version of my pallet counting project, refactored using object-oriented programming (classes, attributes, and methods).

## Concepts Applied

- Object-Oriented Programming (Classes, Attributes, and Methods)

- Exception Handling
  
- Modularization
  
- JSON Data Persistence
  
- Dictionaries and Data Structures


## Features

- Add CHEP pallets
  
- Add broken pallets
  
- Add export pallets
  
- Add slip sheets
  
- Reset all quantities
  
- Display a summary of all pallet counts

## How to Run

1. Clone the repository

```bash
git clone https://github.com/Epic10ka/Pallets-v2
```

2. Navigate to the project folder

```bash
cd Pallets_v2
```

3. Run the application

```bash
python main.py
```
